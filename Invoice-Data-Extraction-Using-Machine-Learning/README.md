# مشروع الطالب: 18- Yousef Meshaal

## 💡 عنوان المشروع:
Invoice-Data-Extraction-Using-Machine-Learning

## 📝 وصف المشروع:
استخراج بيانات الفواتير باستخدام التعلم
هذا المشروع هو نظام ذكاء اصطناعي مصمم لاستخراج البيانات تلقائيًا من الفواتير مختلفة الأشكال. يستخدم تقنيات متقدمة في التعرف البصري على الحروف (OCR) والتعلم العميق (باستخدام نماذج مثل LayoutLM وDonut) لفهم السياق البصري والنصي للمستندات. يتيح النظام للمستخدمين طرح أسئلة بلغة طبيعية (مثل "ما هو رقم الفاتورة؟") ويقوم بتحديد الإجابة وإبرازها مباشرة على صورة الفاتورة، كل ذلك عبر واجهة مستخدم سهلة وبديهية مبنية بـ Gradio.
